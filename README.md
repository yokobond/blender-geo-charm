# 都道府県キーホルダー 3Dモデル生成システム

国土地理院の標高タイルデータを利用して、任意の都道府県の3Dプリント用キーホルダーモデルをBlender上で自動生成します。

> 出典: 国土地理院 地理院タイル (https://maps.gsi.go.jp/development/ichiran.html)

---

## パーツ構成

### 1. 円形土台 (透明アクリル用)
- 直径: 50mm (調整可能: 30〜100mm)
- 厚さ: 2mm
- 素材: 透明アクリル (3Dプリントまたはレーザーカット)
- キーホルダー穴: φ4mm、上端から3mm内側
- **土台の表面に隣接県と海が薄く刻印される**

### 2. メイン地形 (県の立体)
- 標高データに基づく立体地形
- 誇張倍率: 1〜3倍で調整可能
- 最大高さ: 約8mm (倍率2.0x時)
- **土台の上に接着して使用**

### 3. 隣接県の刻印
- 土台表面に0.5mmの薄いレリーフ
- 対象県の周囲の県が淡く表示される

### 4. 海の表示
- 波紋パターンの薄い刻印 (0.5mm)
- 県境外の海域を表現

### 5. 島パーツ (該当する場合)
- 本土から離れた島は別パーツとして生成
- 底面に接着用の凹み (0.3mm)
- 土台の対応位置に貼り付け

## セットアップ

### 前提条件
- Blender 3.6+ (4.x推奨)
- インターネット接続 (標高・県境データ取得時)

### Blender Python への追加パッケージ

```bash
# Blender の Python パスを確認
# (例: /Applications/Blender.app/Contents/Resources/4.0/python/bin/python3.11)

# macOS の場合
/path/to/blender/python/bin/python3.11 -m pip install numpy requests shapely

# Windows の場合
"C:\Program Files\Blender Foundation\Blender 4.0\4.0\python\bin\python.exe" -m pip install numpy requests shapely

# Linux の場合
/path/to/blender/4.0/python/bin/python3.11 -m pip install numpy requests shapely
```

### 実行方法

1. Blender を起動
2. **スクリプティング** ワークスペースに切り替え
3. テキストエディタで `prefecture_keychain.py` を開く
4. `Alt+P` で実行
5. サイドバー (`N` キー) に **「キーホルダー」** タブが表示される
6. パラメータを設定して **「キーホルダーを生成」** をクリック

### コマンドライン実行

```bash
blender --python prefecture_keychain.py
```

自動生成する場合は、スクリプト末尾のコメントを解除:

```python
gen = KeychainGenerator("神奈川県", config={
    'exaggeration': 2.0,
    'resolution': 200,
    'zoom': 10,
})
gen.generate()
```

## パラメータ一覧

| パラメータ | デフォルト | 範囲 | 説明 |
|---|---|---|---|
| 都道府県 | 神奈川県 | 47都道府県 | 対象の都道府県 |
| 標高誇張倍率 | 2.0 | 1.0〜3.0 | 標高を何倍に誇張するか |
| 直径 | 50mm | 30〜100mm | 円形土台の直径 |
| 解像度 | 200 | 50〜500 | メッシュの分割数 |
| ズームレベル | 10 | 8〜14 | 標高タイルの精度 |

### ズームレベルと解像度の目安

| ズームレベル | 地上解像度(約) | 適した用途 |
|---|---|---|
| 8 | 600m | 概観、高速プレビュー |
| 10 | 150m | 標準 (推奨) |
| 12 | 40m | 高精度、小さい県向け |
| 14 | 10m | 最高精度、処理に時間がかかる |

## 3Dプリント手順

### メイン地形パーツ
- **素材**: PLA / PETG / レジン
- **レイヤー高さ**: 0.1〜0.15mm推奨
- **サポート**: 不要 (底面フラット)

### 透明土台
- **素材**: 透明レジン or 透明PETG
- **レイヤー高さ**: 0.1mm (表面品質重視)
- **代替**: レーザーカットした透明アクリル板 (2mm厚)

### 組み立て
1. 透明土台を印刷/カット
2. メイン地形パーツを印刷
3. 島パーツがある場合は別途印刷
4. 地形パーツを土台に接着 (アクリル接着剤 or 瞬間接着剤)
5. 島パーツを土台上の対応位置に接着
6. キーリングを穴に通す

## データソース

- **標高データ**: 国土地理院 地理院タイル 数値標高モデル (DEM10B/DEM5A)
  - URL形式: `https://cyberjapandata.gsi.go.jp/xyz/dem/{z}/{x}/{y}.txt`
  - 256×256 テキスト形式 (カンマ区切り)
  - 単位: メートル

- **県境データ**: dataofjapan/land GeoJSON
  - https://github.com/dataofjapan/land
  - より正確なデータが必要な場合は国土数値情報の行政区域データを使用

## カスタマイズ例

### 色を変更する
`KeychainModelBuilder.build_terrain_mesh()` の `material_color` を変更:
```python
# 雪山風 (白)
material_color=(0.95, 0.95, 0.95, 1.0)

# 抹茶風 (緑)
material_color=(0.3, 0.5, 0.2, 1.0)

# 金属風
material_color=(0.7, 0.65, 0.5, 1.0)
```

### 土台サイズを変更する
```python
config = {
    'diameter_mm': 70.0,      # 70mm直径
    'base_thickness_mm': 3.0, # 3mm厚
    'hole_diameter_mm': 5.0,  # 5mm穴
}
```

### 標高カラーグラデーション
`build_terrain_mesh` に頂点カラーを追加するカスタマイズで、標高に応じた色分けが可能です。

## トラブルシューティング

### 「numpy が必要です」エラー
→ Blender内蔵のPythonにnumpyをインストールしてください (上記セットアップ参照)

### 標高タイルの取得が遅い
→ 解像度やズームレベルを下げてください。初回取得分はキャッシュされます。

### 県の形が表示されない
→ shapely がインストールされているか確認。なければ矩形範囲で代替されます。

### STLエクスポートでエラー
→ `ファイル → エクスポート → STL (.stl)` から手動エクスポートも可能です。
